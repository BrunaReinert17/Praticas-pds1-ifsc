QUESTÕES TEÓRICAS - INTRODUÇÃO AO SWING..

1. Em programação, o que é uma biblioteca?
	R- Refere-se a  uma coleção de recursos usados por um programa de computador,
         você terá acesso a um conjunto de funções que já foram escritas por desenvolvedores.

  2. O que é a biblioteca Swing do Java e qual é o seu propósito?
     R- É uma biblioteca do java que  cria interfaces graficas mais elaboradas um ambiênte heterogêneo com interação mais agradável.
  
  3. Quais são os componentes de interface gráfica de usuário (GUI) fornecidos pela biblioteca Swing para:
			a. Botão - JButton.
			b. Campo de Texto - JTextField
			c. Caixa de Seleção - JCheckBox
			d. Barra de Progresso - JProgressBars
			e. Botão de Opção - JOptionPane
		      f. Caixa de Combinação - JComboBox
		
 4. Quais são os propósitos dos layouts no Swing? E quais são os layouts possíveis?
     R- O propósitos deles é para  definir a posição e o tamanho dos componentes em um contêiner, como uma janela ou painel.São importantes para garantir que os      componetes sejam organizados  de forma coerente e atraente na interface grafica de um usuário.
     Os possiveis layouts são: BorderLayout; FlowLayout; GridLayout; CardLayout; GridBagLayout;

  5. Como é possível fazer a validação de entrada de dados em campos de texto?
    R- É possivel  Restringir a entrada de caracteres: é possivel configurar o campo de texto para aceitar apenas determinados tipos de caracteres, como números,
    letras ou símbolos específicos. Pode ser implementado utilizando a classe 'java.swing.text.AbstractDocument'.

  6. Pesquise e descreva três principais características entre a biblioteca Swing, a biblioteca AWT (Abstract Window Toolkit) e o JavaFX?
R- Elas tem diferenças em termos de caracteristicas e funcionalidades. 
    1° -  Arquitetura e Tecnologia,  Estilo de Programação, Recursos e Capacidades, essas são algumas das principais diferenças entre AWT,Swing e JavaFX.
